## 値オブジェクト

### 値オブジェクトとは

### 値の性質

-   不変である
-   交換(再代入)が可能である
-   等価性によって比較される

> 変数に代入した値を再代入はできても、値そのものを根底から変更する事はできない。
> 値オブジェクトも変更は許容されず、再代入によって交換が可能になる。

## エンティティ

### エンティティとは

### 値オブジェクトとエンティティの違い

-   同一性によって識別されるか

> 人は誕生日(属性)が変わっても**同一人物**として識別される == 同一性が担保されている == エンティティ
> 名前は苗字が変わると**別名**として識別される == 同一性が担保されていない == 値オブジェクト
